Deck Compare — Privacy Policy

Last updated: June 4, 2026

Overview

Deck Compare is a Chrome extension that compares Magic: The Gathering decklists across multiple websites. Your privacy matters — the extension is designed to work entirely on your device with no data collection.

Data we collect

None. Deck Compare does not collect, store, or transmit any personal data to any server.

What the extension does

  • Reads decklist data from publicly available deck pages on supported sites (Moxfield, MTGGoldfish, Archidekt, mtgtop8, Magic-Ville, mtgdecks.net) — only card names and quantities.
  • Fetches card images from the Scryfall API for display purposes.
  • Fetches deck data from supported site APIs to retrieve decklists for comparison.

Local storage

The extension uses Chrome's local storage (chrome.storage.local) to save:

  • Comparison data (two parsed decklists) so the results page can display them.
  • Your Moxfield username and cached deck list, if you choose to enter it, to avoid re-fetching on every popup open.

This data stays entirely on your device. It is never sent to any external server.

Third-party services

The extension makes requests to the following third-party services:

  • Scryfall (api.scryfall.com, cards.scryfall.io) — to fetch card images. Subject to Scryfall's terms.
  • Google Fonts (fonts.googleapis.com, fonts.gstatic.com) — to load display fonts.
  • Supported deck sites — to fetch publicly available deck data via their APIs or pages.

No user data is sent to these services. Only public deck identifiers and card names are used in requests.

Analytics and tracking

Deck Compare does not use any analytics, tracking, cookies, or telemetry of any kind.

Permissions

  • activeTab — to read the decklist from the currently open tab.
  • storage — to save comparison data and Moxfield username locally.
  • Host permissions — to fetch deck data and card images from supported sites and Scryfall.

Changes to this policy

If this policy is updated, the changes will be reflected on this page with an updated date.
